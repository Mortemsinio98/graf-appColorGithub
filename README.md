## **Descripción de la Página**

La página desarrollada es una aplicación web interactiva para la selección y visualización de colores utilizando el modelo RGB. Permite al usuario modificar el color de forma dinámica mediante sliders, campos numéricos y un selector de color, mostrando en tiempo real el resultado en un recuadro visual junto con su código hexadecimal correspondiente.

La interfaz cuenta con un diseño moderno y limpio, apoyado en Bootstrap y estilos personalizados, lo que facilita una experiencia intuitiva y agradable. Todos los controles se encuentran sincronizados, logrando una interacción fluida que combina funcionalidad, claridad visual y un enfoque práctico de aprendizaje basado en la experimentación.
